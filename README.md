# Credit Worthiness Prediction using Machine Learning

## Objective
Predict whether a loan applicant is creditworthy using historical financial data.

## Models Used
- Logistic Regression
- Random Forest

## Best Model
Random Forest

## Performance

| Metric | Logistic Regression | Random Forest |
|--------|--------------------:|--------------:|
| Accuracy | 86.89% | 93.41% |
| ROC-AUC | 0.868 | 0.929 |

## Features
- Person Age
- Person Income
- Employment Length
- Loan Amount
- Interest Rate
- Loan Intent
- Loan Grade
- Home Ownership
- Credit History Length

## Technologies
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
